Authentication

All authenticated endpoints require a Bearer token in the Authorization header. Obtain a token via login, register, or social OAuth — tokens are valid for 7 days.

POST Authenticate with email and password. Returns a Bearer token and user object. Also sets an httpOnly cookie for web browser sessions.
/auth/login
Returns: { success, token, user }

Body

emailstring (required)
User's email address
passwordstring (required)
User's password
Example Request
{
  "email": "fan@example.com",
  "password": "supersecret"
}
Example Response
{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "clx123",
    "name": "TombstonePiledriver",
    "email": "fan@example.com",
    "slug": "tombstone-piledriver-a1b2",
    "avatarUrl": null,
    "isAdmin": false
  }
}
POST Create a new account. Returns a Bearer token and the created user. Username is auto-assigned from a bank of wrestling move names — the user can change it after signup.
/auth/register
Returns: { success, token, user }

Body

namestring (required)
Display name (must be unique, case-insensitive)
emailstring (required)
Email address
passwordstring (required)
Password
Example Response
{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "clx124",
    "name": "SwantonBomb",
    "email": "new@example.com",
    "slug": "swanton-bomb-z9y8"
  }
}
GET Initiate Google OAuth. For the mobile app, pass ?platform=app — after Google authenticates the user, the callback will redirect to the poisonrana:// deep link instead of the web homepage.
/auth/google?platform=app
Returns: 302 redirect → Google sign-in → poisonrana://auth?token=... (app) or / (web)

Query

platformstring
"app" to receive token via deep link (poisonrana://auth?token=...). Omit for web flow.
GET Initiate Facebook OAuth. Same platform=app deep-link behaviour as Google.
/auth/facebook?platform=app
Returns: 302 redirect → Facebook sign-in → poisonrana://auth?token=... (app) or / (web)

Query

platformstring
"app" to receive token via deep link. Omit for web flow.
GETAuth Returns the currently authenticated user.
/auth/me
Returns: { user }

Events

Browse, search and retrieve professional wrestling events.

GET List all events with optional filtering, search and pagination.
/events
Returns: Event[]

Query

pagenumber
Page number (default: 1)
limitnumber
Results per page, max 50 (default: 20)
qstring
Search by event title
promotionstring
Filter by promotion (e.g. WWE, AEW)
typestring
Filter by event type (e.g. PPV, TV)
sortstring
Sort order: date_desc (default) | date_asc

Rankings

Ranked lists using the Poison Rana Bayesian weighted rating formula: (v/(v+m)) × R + (m/(v+m)) × C — where v = review count, m = minimum review threshold, R = event average, C = global average.

GET Top events ranked by Bayesian weighted rating.
/rankings/events
Returns: RankedEvent[]

Query

promotionstring
Filter to a specific promotion
limitnumber
Max results, up to 100 (default: 50)
Example Response
{
  "data": [
    {
      "id": "clx789",
      "title": "WrestleKingdom 17",
      "slug": "wrestlekingdom-17",
      "date": "2023-01-04T00:00:00.000Z",
      "promotion": "NJPW",
      "posterUrl": "https://...",
      "reviewCount": 88,
      "averageRating": 4.71,
      "bayesianScore": 4.643
    }
  ],
  "error": null,
  "meta": {
    "total": 50
  }
}

My Profile

Endpoints for the authenticated user's own data.

GETAuth Get the authenticated user's profile with activity counts.
/me
Returns: User with counts
PATCHAuth Update the authenticated user's profile.
/me
Returns: Updated user

Body

namestring
Display name
avatarUrlstring
Avatar image URL (null to remove)
favoritePromotionstring
Favourite promotion short name
GETAuth Get the authenticated user's watchlist with pagination.
/me/watchlist
Returns: WatchlistItem[]

Query

pagenumber
Page number (default: 1)
limitnumber
Results per page, max 50 (default: 20)
POSTAuth Add an event to the watchlist.
/me/watchlist
Returns: WatchlistItem

Body

eventIdstring (required)
Event ID to add
DELETEAuth Remove an event from the watchlist.
/me/watchlist
Returns: { removed: true }

Query

eventIdstring (required)
Event ID to remove
GETAuth Get a paginated activity feed of reviews from users you follow.
/me/feed
Returns: Review[]

Query

pagenumber
Page number (default: 1)
limitnumber
Results per page, max 50 (default: 20)

Interactions

Submit ratings, predictions, and follow other users.

POSTAuth Submit or update a star rating for a match.
/matches/:id/rate
Returns: { average, ratingCount }

Body

ratingnumber (required)
Rating from 0.25 to 5
POSTAuth Submit a prediction for a match. Set predictedWinnerId to null to clear your pick. Locked once the event starts or a winner is marked.
/predictions
Returns: Prediction | { deleted: true }

Body

matchIdstring (required)
Match ID
predictedWinnerIdstring | null (required)
Participant wrestler ID, or null to clear
GET Get community prediction stats for a match.
/predictions
Returns: { stats: [{ winnerId, count, percentage }], total }

Query

matchIdstring (required)
Match ID
POSTAuth Toggle follow/unfollow a user. Returns the new follow state.
/follow
Returns: { followed: boolean }

Body

targetUserIdstring (required)
User ID to follow or unfollow
